# Project Vortex API Documentation

This folder is reserved for API schemas and interactive endpoints references.
目前 (currently), FastAPI exposes a local development endpoint:

- **Swagger UI**: [http://localhost:8000/docs](http://localhost:8000/docs)
- **ReDoc**: [http://localhost:8000/redoc](http://localhost:8000/redoc)

## Generating API Definitions
If you establish external SDKs or separate frontend repos, consider exporting OpenAPI definitions here:
`docs/api/openapi.json`
`docs/api/postman_collection.json`
